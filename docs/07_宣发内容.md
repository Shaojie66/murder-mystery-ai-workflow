## 平台策略

### B站/YouTube

**目标**：建立专业权威，吸引深度用户

**内容方向**：
- "我用AI 11个月做出一整套剧本杀"
- "剧本杀创作教程系列"
- "独立创作者如何用AI工具"

**视频结构**：
```
1. 开头（30秒）：钩子 + 悬念
2. 背景（1分钟）：我是谁 + 为什么做
3. 干货（10分钟）：创作流程 + 技巧
4. 结尾（1分钟）：引导关注 + 下期预告
```

**关键指标**：完播率 > 40%

---

### 小红书

**目标**：种草转化，引流私域

**内容方向**：
- 产品展示（精美图）
- 创作过程（behind the scenes）
- 玩家体验（UGC）
- 教程技巧（干干货）

**图文模板**：
```
封面：美图 + 标题
正文：
- 开头：痛点/共鸣
- 中间：产品/内容展示
- 结尾：行动号召（评论区/链接）
```

**短视频模板**：
```
0-3秒：钩子（你绝对想不到...）
3-10秒：冲突/悬念
10-30秒：揭晓/解决方案
```

---

### 公众号

**目标**：深度内容，粉丝沉淀

**文章方向**：
- 创作复盘（长文）
- 行业洞察（剧本杀市场）
- 工具测评（AI工具）
- 粉丝问答

**发布节奏**：
- 周二/周五发布
- 配合节点（节假日、热点）

---

### 抖音/视频号

**目标**：泛流量，快速扩散

**内容方向**：
- 创作过程快剪
- 有趣片段/梗
- 玩家反应
- 挑战/互动

**爆款公式**：
```
开头：3秒定律（必须留住人）
内容：反转/惊喜/共鸣
结尾：互动引导（评论/分享）
```
