# 06_印刷生产

> 从打样到量产，确保印刷品质符合预期。

---

## 印刷厂选择

### 筛选标准

| 维度 | 要求 |
|------|------|
| 样品质量 | 能提供同类产品样品 |
| 最小起订量 | ≤100盒 |
| 交货周期 | ≤2周 |
| 价格 | 市场均价或更低 |
| 文件格式 | 支持PDF/AI |

### 推荐印刷厂类型

1. **淘宝专业印刷店**
   - 优点：门槛低、可小批量、价格透明
   - 缺点：质量参差不齐

2. **阿里爸爸找工厂**
   - 优点：可筛选有完整生产线的工厂
   - 缺点：起订量可能较高

3. **本地图文店**
   - 优点：沟通方便、可看样品
   - 缺点：价格较高

### 比价流程

```
1. 至少联系3家印刷厂
2. 发送完整文件包（盒、说明、卡牌等）
3. 询问报价（含税、含运费）
4. 要求看同类产品样品
5. 综合评估选择
```

## 打样流程

### 第一批：盒子和说明书

```
目的：验证颜色、材质、尺寸
数量：1-3套
时间：3-5天
重点检查：
□ 颜色是否偏色
□ 纸张克数是否合适
□ 覆膜效果（哑光/亮膜）
□ 折痕位置是否准确
□ 尺寸是否合适
```

### 第二批：卡牌和面板

```
目的：验证卡片质量
数量：1套完整+10套卡牌
时间：3-5天
重点检查：
□ 卡牌硬度/挺度
□ 覆膜效果
□ 切边是否整齐
□ 尺寸一致性
```

### 第三批：量产确认样

```
目的：确认最终效果
数量：5套
时间：1周
全部检查通过后，开始量产
```

## 量产流程

### 文件准备

| 文件 | 格式要求 |
|------|----------|
| 盒子 | PDF/AI，CMYK，3mm出血 |
| 说明书 | PDF，CMYK，3mm出血 |
| 角色本 | PDF，CMYK |
| 卡牌 | PDF，CMYK |
| 面板 | PDF/AI，CMYK |

### 生产排期

| 阶段 | 时间 | 内容 |
|------|------|------|
| 确认文件 | Day 1 | 发送最终文件 |
| 排版印刷 | Day 2-5 | 印刷 |
| 覆膜裁切 | Day 6-8 | 后加工 |
| 包装组装 | Day 9-10 | 包装 |
| 质检发货 | Day 11-12 | 质检+发货 |

## 交稿格式指南

### 通用要求

- 色彩模式：CMYK
- 分辨率：300dpi
- 出血：3mm
- 字体：嵌入或转曲

### 各物料要求

| 物料 | 尺寸 | 备注 |
|------|------|------|
| 外盒 | 展开尺寸+出血 | 刀版图需向印刷厂要 |
| 说明书 | A4或自定义 | 单页/多页 |
| 角色本 | A5 |骑马钉/胶装 |
| 线索卡 | 57×87mm（扑克牌尺寸） | 需圆角 |
| 面板 | A4/自定义 | 覆硬纸板 |

### 常用卡牌尺寸参考

```
标准扑克牌：57×87mm
剧本杀线索卡：60×90mm / 70×100mm
厚卡牌：+0.3-0.5mm厚度
```

## 质检标准

### 量产抽检

```
抽检比例：5%
抽检项目：
□ 颜色一致性
□ 尺寸准确性
□ 切边整齐度
□ 覆膜完整性
□ 数量准确性
```

### 不良品处理

- 拍照留存
- 记录不良数量
- 与印刷厂协商补印或折扣

## 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 颜色偏暗 | RGB转CMYK丢失 | 使用专业CMYK文件 |
| 卡牌粘连 | 覆膜质量问题 | 覆膜前确保干燥 |
| 尺寸偏差 | 裁切误差 | 留足出血、设计时微调 |
| 起订量太高 | 工厂限制 | 找支持小批量厂家 |
