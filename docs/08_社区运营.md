# 08_社区运营

> 维护玩家社群，建立反馈闭环，规划扩展包。

---

## 社群搭建

### 平台选择

| 平台 | 优点 | 缺点 | 适合人群 |
|------|------|------|----------|
| 微信群 | 私密、方便 | 难扩展、不便于管理 | 核心玩家 |
| QQ群 | 功能多、易管理 | 年轻人少 | 技术型玩家 |
| Discord | 功能强大、国际化 | 门槛高 | 海外玩家 |

### 社群结构

```
总群（公开）
├── 核心玩家群（VIP）
├── DM交流群
└── 创作交流群（可选）
```

### 入群门槛

- 购买过剧本杀（验单）
- 填写问卷（了解程度）
- 设置群规

## 玩家群运营

### 日常运营

| 项目 | 频率 | 内容 |
|------|------|------|
| 早安/晚安 | 每天 | 问候 + 话题引导 |
| 固定活动 | 每周 | 线上/线下局 |
| 内容分享 | 每天 | 行业资讯/创作日常 |

### 活动策划

| 活动类型 | 说明 | 频率 |
|----------|------|------|
| 线上开黑 | 语音/视频开局 | 每周 |
| 线下聚会 | 本地玩家组队 | 每月 |
| 创作比赛 | 剧本杀创作赛 | 每季度 |
| 节日活动 | 节假日特别活动 | 节日 |

### 激励体系

```
活跃玩家
├── 优先内测资格
├── 专属客服
├── 优惠购买
└── 署名感谢

核心创作者
├── 加入创作团队
├── 分成比例
└── IP署名
```

## 反馈闭环

### 反馈收集

| 渠道 | 方式 |
|------|------|
| 问卷表单 | 腾讯问卷/问卷星 |
| 微信群 | 直接收集 |
| 邮箱 | 官方邮箱 |
| 小红书 | 评论区 |

### 反馈分类

| 类型 | 处理方式 | 响应时间 |
|------|----------|----------|
| BUG/穿帮 | 记录、修复 | 1周内 |
| 建议 | 评估、回复 | 3天内 |
| 投诉 | 处理、跟进 | 1天内 |

### 闭环流程

```
收集 → 分类 → 处理 → 反馈 → 迭代
```
