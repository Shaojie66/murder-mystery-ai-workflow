### 技巧：风格锚定

为确保6个角色同一画风，在每个Prompt里加：

```
统一的风格前缀：
古风水墨, 工笔画质感, 细腻线条,
色彩饱和度适中, 东方美学, 4K高清
```

### 技巧：负面Prompt

```
不要出现：
现代元素, 卡通风格, 过度饱和,
西方奇幻, 比例失真, 模糊, 噪点
```

## 三版迭代法

```
版本1：基础版
└─ Prompt：[角色描述] + [风格前缀]
   产出：10张候选
   目的：确认基本构图和气质

版本2：迭代版
├─ 选择：最好的1-2张
├─ 调整Prompt：加强/修改不满意的细节
└─ 产出：10张候选
   目的：优化服饰/神态/场景

版本3：终版
├─ 选择：最满意的1张
├─ 高分辨率生成
└─ 输出：可印刷分辨率
```
