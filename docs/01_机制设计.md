# 阶段一：机制设计

## 两种方式

---

### 方式A：Gemini 深度研究（原创机制）

```
你是一个桌游机制设计师。请深度研究以下机制概念：
[输入你的核心想法]

研究维度：
1. 博弈论分析 - 这个机制是否会产生有趣的博弈？
2. 平衡性检查 - 六种角色是否都能存活？
3. 策略深度 - 是否有足够的策略空间？
4. 执行障碍 - 规则是否过于复杂？
5. 情绪曲线 - 玩家体验如何递进？

输出：
- 机制核心公式
- 平衡性建议
- 潜在问题及解决方案
- 可参考的桌游案例
```

---

### 方式B：规则书改编（适合快速启动）

```
你是一个桌游改编专家。请分析以下规则书，
并设计一个全新的剧本杀世界观和故事线：

[粘贴规则书内容]

要求：
1. 保留核心机制（动态克制/压力值/资源管理等）
2. 匹配一个新的故事背景（古代/现代/奇幻/科幻）
3. 设计6个角色，贴合机制定位
4. 核心冲突必须与机制深度绑定
5. 输出完整的游戏大纲（3-5页）

输出格式：
- 世界观设定（1页）
- 核心冲突线（1页）
- 角色定位表（1页）
- 事件时间线（1页）
- 机制-叙事绑定说明（1页）
```

---

## 机制类型参考

| 类型 | 特点 | 适合的叙事主题 |
|------|------|---------------|
| 动态克制 | 兵种/技能随环境变化 | 战争、帮派、资源争夺 |
| 压力值 | 心理压力替代生命值 | 悬疑、恐怖、审讯 |
| 暗拍机制 | 隐藏出价/行动 | 商战、宫廷斗争 |
| 身份推理 | 隐藏身份+技能 | 谍战、帮派卧底 |
| 区域控制 | 地图+占领 | 领地争夺、跑团 |

---
