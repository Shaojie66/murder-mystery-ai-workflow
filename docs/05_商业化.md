## 成本核算

### 印刷成本明细

| 项目 | 单价 | 数量 | 小计 |
|------|------|------|------|
| 外盒 | ¥3-5 | 1 | ¥3-5 |
| 说明书 | ¥1-2 | 1 | ¥1-2 |
| 角色本（a本） | ¥0.5/页 | 约20页×6 | ¥60 |
| 线索本（b本） | ¥0.3/页 | 约15页×6 | ¥27 |
| 角色面板 | ¥0.3 | 6 | ¥1.8 |
| 线索卡 | ¥0.1 | 约30张 | ¥3 |
| 人物小卡 | ¥0.2 | 6 | ¥1.2 |
| 地图/板块 | ¥1-2 | 1 | ¥1-2 |
| 包装材料 | ¥2-3 | - | ¥2-3 |
| **合计** | | | **¥100-130** |

### 单位成本计算

```
单位成本 = 印刷成本 + 包装 + 物流预估
         = ¥130 + ¥10 + ¥10
         = ¥150
```

## 销售渠道

### 线上渠道

| 渠道 | 优点 | 缺点 | 适合 |
|------|------|------|------|
| 淘宝/闲鱼 | 流量大、操作简单 | 分成高、竞争激烈 | 初期快速出货 |
| 微店/有赞 | 自主定价、无分成 | 需自引流 | 私域销售 |
| 小红书商店 | 年轻用户多 | 审核严格 | 种草转化 |

### 线下渠道

| 渠道 | 方式 | 分成比例 |
|------|------|----------|
| 剧本杀店直供 | 联系店主寄售 | 5-5分成或买断 |
| 桌游吧合作 | 铺货销售 | 4-6分成 |
| 展会销售 | 参加剧本杀展会 | 现场售卖 |

### 定价建议

```
新手/试水：¥399-499（低利润快速跑量）
标准版：¥599-699（推荐，利润适中）
珍藏版：¥799-999（高利润，限量的噱头）
```

## 风险管理

| 风险 | 应对 |
|------|------|
| 首版印刷质量差 | 先打样，确认后再量产 |
| 滞销积压 | 首批少印（50-100盒），验证后再加印 |
| 盗版问题 | 添加防伪标识，保留版权证明 |
| 玩家差评 | 测试阶段充分验证，提前准备FAQ |
