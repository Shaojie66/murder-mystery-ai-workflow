# 访谈招募计划

**目标**：招募 10 名剧本杀创作者参与访谈
**目标配比**：业余爱好者 5 名 + 职业/工作室创作者 5 名
**访谈时长**：30-45 分钟/人
**激励**：礼品卡 ¥50-100 或等值礼物

---

## 招募画像

### 画像 A：业余爱好者（5名）

| 条件 | 说明 |
|------|------|
| 创作经验 | 至少完成过 1 个剧本杀（已发行或内测） |
| 协作经历 | 有过或没有协作经历均可（非必要条件） |
| 技术背景 | 非程序员优先（验证引导式界面需求） |
| 渠道 | 小红书、知乎、豆瓣剧本杀小组 |

### 画像 B：职业/工作室创作者（5名）

| 条件 | 说明 |
|------|------|
| 创作经验 | 至少完成过 3 个剧本杀，至少 1 个已商业化 |
| 协作经历 | 有过真实协作经历（必要条件） |
| 团队背景 | 2-5 人工作室优先 |
| 渠道 | 剧本杀论坛、微信创作者群、展会圈 |

---

## 招募渠道

### 渠道 1：小红书

**优势**：剧本杀爱好者聚集，内容以经验分享为主
**操作方式**：
1. 搜索"剧本杀创作"、"剧本杀作者"、"剧本杀工作室"相关笔记
2. 找到有创作经历的博主，私信联系
3. 评论区招募：在热门帖子下留言招募

**模板私信**：
```
你好！看到你对剧本杀创作很有经验（你的[某篇]笔记写得很好）。

我正在做一个关于剧本杀创作工具的用户研究，想了解创作者的真实工作方式。正在找有创作经验的朋友做 30-45 分钟的访谈，线上进行，会有 ¥50-100 的礼品感谢。

有兴趣吗？大概需要 30-45 分钟，聊聊你创作剧本杀的过程和遇到的挑战。
```

### 渠道 2：知乎

**优势**：专业讨论为主，用户愿意深度参与
**操作方式**：
1. 在"剧本杀"、"桌游"、"剧本创作"相关问题下寻找回答者
2. 关注"剧本杀创作者"话题下的活跃用户
3. 私信联系

**模板私信**：同上

### 渠道 3：剧本杀社群/微信群

**优势**：精准触达目标用户
**操作方式**：
1. 加入剧本杀创作者群（通过朋友介绍或知识星球）
2. 在群内发布招募信息
3. 请群主/管理员帮忙推荐

**群内招募文案**：
```
【用户访谈招募】30-45分钟，了解剧本杀创作者的真实需求

我是做剧本杀创作工具的产品经理，最近在做用户研究，想了解大家在创作过程中遇到的真实问题。

需要找 10 名有创作经验的朋友做访谈：
- 业余爱好者（完成过至少 1 个剧本杀）
- 职业创作者/工作室（至少 3 个作品，至少 1 个已商业化）

访谈时间：30-45 分钟，线上进行
感谢：¥50-100 礼品卡

有兴趣的朋友请私信我，或者在下面留言～谢谢！
```

### 渠道 4：豆瓣小组

**优势**：有剧本杀专属小组，讨论深入
**操作方式**：
1. 加入"剧本杀"、"桌游创作"相关小组
2. 发布招募帖子
3. 主动联系回复过的用户

### 渠道 5：剧本杀展会/活动

**优势**：面对面接触，信任度高
**操作方式**：
1. 关注各地剧本杀展会信息（全国剧本杀剧本展等）
2. 现场接触创作者，留联系方式
3. 展会后邀约访谈

## 时间表

```
第 1 周：渠道铺设
  Day 1-2：准备招募文案、各渠道账号准备
  Day 3-5：小红书、知乎、豆瓣发布招募信息

第 2 周：筛选与预约
  Day 1-3：收集响应、筛选符合画像的对象
  Day 4-5：发送确认邮件/日历邀请

第 3 周：执行访谈（可与第 2 周重叠）
  每天安排 1-2 人访谈
  当天整理访谈记录

第 4 周：补充招募 + 收尾
  如人数不足，补充招募
  汇总所有访谈记录
```

## 筛选问卷（预筛选）

在正式邀约访谈前，用简短问卷确认是否符合画像：

**问卷问题**（5 题，2 分钟完成）：

1. 你目前创作剧本杀的主要身份是？
   - 业余爱好者
   - 独立创作者（部分时间）
   - 职业创作者/全职
   - 工作室/团队成员

2. 你已完成（包括已发行和内测）的剧本杀数量是？
   - 0 个
   - 1-2 个
   - 3-5 个
   - 5 个以上

3. 你是否有与其他创作者协作完成剧本杀的经历？
   - 有（继续回答）
   - 没有（跳至 Q5）

4. 协作中遇到的最大挑战是什么？（多选）
   - 文件传输/版本管理
   - 分工与协调
   - 沟通效率
   - 其他：___

5. 如果有一个在线协作工具可以解决剧本杀创作中的协作问题，你有兴趣了解吗？
   - 非常有兴趣
   - 有些兴趣
   - 一般
   - 不太有兴趣

**筛选标准**：
- 画像 A：Q1 选 A/Q2 ≥ 1，Q5 选 A/B
- 画像 B：Q1 选 C/D，Q2 ≥ 3，Q3 = 有，Q4 有具体回答

## 备选方案

如果招募困难：

| 问题 | 备选方案 |
|------|---------|
| 渠道响应率低 | 增加激励（¥100 → ¥150），或提供产品优先使用权 |
| 职业创作者难找 | 通过剧本杀展会、创作者社群找，关键人物推荐 |
| 时间难以协调 | 提供多个时间段选项，线上线下均可 |
| 人数不足 10 人 | 降至 6 人（3+3），但需注明样本局限性 |
