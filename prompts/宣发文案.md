# 宣发文案 Prompt

> 适用于全平台内容创作

---

## Prompt 1：产品种草文案（小红书）

```
请为剧本杀创作**小红书种草文案**：

产品：【剧本杀名称】
特点：【核心卖点】
目标用户：【目标人群】

要求：
1. 标题要有**钩子**（30字内），吸引点击
2. 开头要有**共鸣点**，让用户感同身受
3. 中间展示**产品亮点**，用具体细节说服
4. 结尾有**行动号召**，引导购买
5. 风格：年轻、亲切、有代入感
6. 加上合适的**话题标签**（5-8个）

输出格式：
```markdown
【标题】

【正文】

#话题1 #话题2 #话题3 ...
```
```

## Prompt 2：B站视频脚本

```
请为剧本杀创作**B站长视频脚本**：

主题：【视频主题，如：用AI做剧本杀】
时长：15-20分钟
目标：建立专业权威，吸引深度用户

结构要求：
1. **开头（30秒）**：强钩子 + 悬念，留住观众
2. **背景（1分钟）**：我是谁 + 为什么做这个
3. **干货（10-15分钟）**：创作流程 + 技巧，干货满满
4. **结尾（1分钟）**：总结 + 引导关注 + 下期预告

风格要求：
- 口语化、有感染力
- 适当加入个人情绪
- 保持专业度

输出格式：完整脚本，包含时间节点和画面描述
```

## Prompt 3：公众号深度文章

```
请为剧本杀创作**公众号深度文章**：

主题：【文章主题】
目的：粉丝沉淀、建立专业形象
字数：2000-2500字

结构：
1. **开头**：痛点/故事引入（200字）
2. **正文**：3-4个章节，每章500-600字
3. **结尾**：总结 + 互动引导

可选角度：
- 创作复盘
- 行业洞察
- 工具测评
- 经验分享

要求：
- 有深度、有观点
- 案例丰富、细节充分
- 逻辑清晰、可读性强

输出格式：Markdown，章节分明
```

## Prompt 4：抖音短视频脚本

```
请为剧本杀创作**抖音/视频号短视频脚本**：

主题：【内容主题】
时长：15-30秒
目的：流量扩散、吸引关注

脚本要求：
1. **0-3秒**：强钩子，必须留住人（反转/悬念/共鸣）
2. **3-10秒**：内容展开，制造期待
3. **10-30秒**：高潮/揭晓，引导互动

风格：
- 快节奏、有爆点
- 视觉冲击强
- 音乐/音效配合
- 结尾留悬念或引导评论

输出格式：
```markdown
【时长】画面/内容/台词/音效

[00:00-00:03]
画面：
台词：
音效：

[00:03-00:10]
...
```
```

## Prompt 6：产品详情页

```
请为剧本杀创作**淘宝/微店产品详情页**：

产品：【剧本杀名称】
定价：【价格】
特点：【核心卖点】

详情页结构：
1. **主图区域**：吸引眼球的封面图描述
2. **痛点切入**：目标用户的困扰
3. **产品介绍**：详细介绍剧本杀特点
4. **内容展示**：包含什么（角色本、线索卡等）
5. **买家秀/评价**：引导信任
6. **购买引导**：行动号召

要求：
- 视觉化描述（方便设计师制作）
- 卖点清晰、层次分明
- 转化导向

输出格式：Markdown + 视觉描述
```
