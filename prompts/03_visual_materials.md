# 视觉物料 Prompt

> 适用于 即梦（Jimeng）/ 海螺AI / Midjourney

---

## 设计原则

1. **国风 + 电影感** — 色调克制，构图讲究，区别于日系二次元
2. **氛围优先** — 人物图要传递情绪，不只是美型
3. **可落地** — 提示词要能实际生成，不是艺术描述

---

## Q1：角色立绘

```
【角色背景设定】

请为角色「{角色名}」生成一幅角色立绘提示词。

## 角色信息
- 年龄感：青年/中年/老年
- 气质关键词：（从背景设定提取2-3个）
- 身份标签：
- 核心情绪：（从背景设定提取）

## 构图要求
- 景别：胸像/全身/特写
- 视角：平视/俯视/仰视
- 与观众的距离感：亲近/疏离/压迫

## 风格约束
- 色调：（从剧本氛围提取，如：灰蓝=冷峻，暗红=血腥，青绿=山水）
- 光影：自然光/戏剧光/剪影
- 不要元素：（明确不要的，如：无笑面、无日式美型、无韩式滤镜）

## 输出格式
```markdown
## {角色名}立绘提示词（英文）

主体描述：（具体的外貌特征，不是"帅哥美女"）
场景/背景：（一个具体的场景片段，而非抽象背景）
光影描述：（主光方向、色温、强度）
色调关键词：（3-5个）
风格标签：（如：cinematic, naturalistic, dark fantasy）

## 变体1：（另一个角度/情绪）
## 变体2：（另一个角度/情绪）
```

提示词原则：
- 用具体名词，不用形容词（"苍白的消瘦的脸"而非"病态美感"）
- 用场景片段而非情绪词（"雨夜独自站在码头"而非"孤独感"）
- 英文为主，关键词+细节+风格标签
```

## 构图要求
- 景别：全景/中景/特写
- 时间：白天/黄昏/夜晚
- 天气/光线：

## 风格约束
- 色调主调：（3个以内关键词）
- 不要元素：

## 输出格式
```markdown
## 场景名：{场景名}

主体：（画面中心是什么，具体描述）
环境：（背景细节，2-3个层次）
氛围细节：（光/雨/雾/影）
色调：（3个关键词）
风格标签：

## 变体1：（另一个时间段）
## 变体2：（另一个情绪角度）

## Q3：线索卡底图

```
请为剧本设计{线索卡数量}张线索卡底图。

## 设计规范
- 尺寸：标准卡牌比例（63mm x 88mm 或 59mm x 86mm）
- 边距：四周留3mm以上白边
- 分辨率：300dpi

## 风格要求
- 每张底图对应不同类型的线索：
  - 实物类（物品/文件）：写实风格
  - 记忆类（场景/人物）：半抽象
  - 证据类（细节/文字）：档案风格
- 色调应与角色立绘有差异，避免混淆

## 每张卡牌需输出
- 底图提示词（英文）
- 适合放置的文字位置建议
- 颜色方案

格式：表格，每张卡一列
```
