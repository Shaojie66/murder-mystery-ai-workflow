# 机制研究 Prompt

> 适用于 Gemini 3 Pro 深度研究模式

---

## Prompt 1：原创机制构思

```
你是一位桌游机制设计师。请为一款6人剧本杀设计一套**原创游戏机制**。

要求：
1. 机制要有**独特性**，区别于市面上常见的剧本杀
2. **可玩性强**，6人游戏节奏流畅
3. **平衡性**，每个角色都有独特价值
4. **主题契合**，机制与故事背景高度统一

请提供：
- 机制名称
- 核心机制详解（3-5个主要机制）
- 每个角色的特殊能力
- 游戏流程概述
- 机制平衡性说明

输出格式：结构化Markdown
```

---

## Prompt 2：规则书改编分析

```
请分析以下规则书，提取可改编为剧本杀的核心机制：

【粘贴规则书内容】

分析维度：
1. **核心机制**：哪些机制可以保留？
2. **故事框架**：如何构建6人剧本故事？
3. **角色转换**：如何将桌游角色转为剧本杀角色？
4. **改编难点**：可能存在哪些问题？
5. **建议方案**：如何优化改编？

输出格式：结构化Markdown
```

---

## Prompt 3：机制平衡性验证

```
请验证以下剧本杀机制的平衡性：

【粘贴机制详情】

验证维度：
1. **角色强度**：是否存在明显过强/过弱的角色？
2. **信息不对称**：信息差设计是否合理？
3. **胜利条件**：凶手与非凶手获胜概率是否接近50%？
4. **游戏时长**：预计时长是否合理（3-4小时）？
5. **改进建议**：列出具体平衡调整建议

输出格式：结构化Markdown + 平衡性评分(1-10)
```

基于以下机制，提出3-5个**扩展包**创意：
