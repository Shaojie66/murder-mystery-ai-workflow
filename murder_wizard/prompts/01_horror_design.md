# 恐怖设计 Prompt — 恐怖本

> 适用于 恐怖本（horror）类型剧本杀
> 核心：恐怖氛围、悬疑营造、心理压迫

---

## 恐怖本设计原则

1. **氛围即产品** — 恐怖感来自整体氛围而非单一jump scare
2. **未知即恐惧** — 想象力比实际更可怕
3. **压迫即动力** — 玩家需要在压力下做决定
4. **安全即底线** — 恐怖有边界，不能越过玩家的舒适底线

---

## Q1：恐怖基因验证

```
【brief.md 核心内容】
{{brief_content}}

在设计任何具体内容之前，先回答：

1. 这个剧本的**恐怖核心**是什么？
   - 主要恐怖类型是什么？（血腥/悬疑/心理/超自然/社会...）
   - 恐怖是**瞬间冲击**还是**持续压迫**？
   - 玩家在游戏中会感到"安全"吗？（恐怖本需要喘息空间）

2. **氛围构建验证**：
   - 是否有足够的**氛围铺垫**（不能上来就吓人）？
   - 是否有**渐进式升级**（越来越恐怖）？
   - 恐怖元素是否有**节奏感**（不能一直维持最高紧张度）？

3. 最危险的恐怖设计失误：
   - 是否会**过度血腥**（为了恐怖而恐怖）？
   - 是否有**廉价jump scare**（突然大声/突然出现）？
   - 恐怖是否**脱离故事**（为了吓人而不顾逻辑）？

请给出：
- 恐怖基因评分（1-10，附理由）
- 3个以内**必须解决**的恐怖设计问题
```

## Q3：悬念结构验证

```
请验证以下恐怖设计的悬念结构：

【完整恐怖设计】
{{mechanism_content}}

验证维度：

### 悬念铺垫验证
- 恐怖是否有**充分铺垫**（不能凭空出现）？
- 是否有**悬念递进**（越来越想知道真相）？
- 悬念是否**回应故事**（不是为悬疑而悬疑）？

### 信息控制验证
- 玩家是否知道**太多/太少**？（平衡点在哪里）
- 是否有**分步揭示**机制（一点一点揭露真相）？
- 是否有**误导性信息**？（假线索/假恐惧）

### 心理压迫验证
- 玩家是否会感到**心理压力**（不是身体危险而是心理层面）？
- 是否有**两难选择**（每个选择都有代价）？
- 玩家是否能**做出有意义的选择**？

### 结局恐惧验证
- 结局是否足够**震撼/意外**？
- 是否有**余韵恐惧**（走出房间后还感到害怕）？
- 是否有**希望元素**（不至于让玩家心理受伤）？

对于每个问题，给出：
- 问题描述
- 严重程度（P1/P2/P3）
- 具体修复建议
```

## Q4：真相设计

```
基于以下已完成的恐怖设计，设计真相揭露方案：

【完整恐怖设计 + 悬念报告】
{{mechanism_and_balance}}

真相设计要求：
- 真相必须有**情感冲击**（不是"哦原来是这样"）
- 真相必须有**反转**（玩家预期的vs实际发生的）
- 真相必须有**余味**（玩家会想"如果是我会怎么做"）

真相类型参考：
- 命运型：一切都是注定的
- 选择型：是你的选择导致了悲剧
- 误解型：你一直误解了某人/某事
- 循环型：历史在重演

每个真相方案包含：
- 表面真相（玩家以为的）
- 真正真相（实际发生的）
- 揭露方式（如何在游戏中逐步揭示）
- 情感冲击点

格式：每个方案一节，400字以内
```
