# 情感设计 Prompt — 情感本

> 适用于 情感本（emotion）类型剧本杀
> 核心：角色情感、人物关系、沉浸式体验

---

## 情感本设计原则

1. **情感弧线即产品** — 每个角色的情感变化是剧本的灵魂
2. **关系即机制** — 人物关系本身就是游戏机制（爱恨情仇）
3. **沉浸即胜利** — 玩家是否能代入角色、产生情感共鸣是成功标准
4. **选择即代价** — 情感抉择必须有两难性

---

## Q1：情感定位验证

```
【brief.md 核心内容】
{{brief_content}}

在设计任何具体内容之前，先回答：

1. 这个剧本的**情感核心**是什么？
   - 玩家最核心的情感体验是什么？（如：失去、愧疚、救赎、误解、错过...）
   - 这个情感是否有**普遍性**？（能让不同背景的玩家都产生共鸣）

2. **关系网络**验证：
   - {{char_count}}个角色之间有多少条情感关系线？
   - 每条线的情感张力是否足够？（不是"认识"，是"有故事"）
   - 是否存在**核心关系对**？（整个关系网围绕哪1-2对展开）

3. 最危险的情感设计失误：
   - 角色是否过于"完美"或"可预测"？
   - 情感冲突是否**被迫**而非自然产生？
   - 是否有"尬点"（玩家会出戏的时刻）？

请给出：
- 情感定位评分（1-10，附理由）
- 3个以内**必须解决**的情感设计问题
```

## Q2：角色情感设计

```
基于以下已验证的情感定位，设计{{char_count}}位角色的情感架构：

【Q1 验证结果】
{{q1_result}}

## 必须包含

### 2.1 核心情感锚点
每位角色必须有：
- **一个最深的情感伤口**（过去经历造成的）
- **一个当前最强烈的情感诉求**（游戏中想要什么）
- **一个会在游戏中被动摇的信念**（情感弧线的起点）

### 2.2 关系矩阵
| 角色 | 与R1的情感关系 | 与R2的情感关系 | 关系性质 | 情感张力 |
|------|---------------|---------------|----------|----------|
| R1 | — | | | |
| R2 | | — | | |
（填充{{char_count}}×{{char_count}}矩阵，对角线为空）

### 2.3 关键情感场景（每个角色至少2个）
- **第一幕**：玩家读本后第一个情感冲击点是什么？
- **中盘**：在什么情况下角色会情绪崩溃/爆发/做出意外选择？
- **结局前**：角色必须面对的最后一个情感抉择是什么？

### 2.4 沉浸触发器
- 哪些**具体细节**能让玩家瞬间代入？（避免泛泛的"他很悲伤"）
- 角色的那些**小动作/口头禅/习惯**能让人物更真实？
- 是否有**物件/场景**可以作为情感记忆的载体？

## Q3：情感一致性验证

```
请验证以下情感设计的内部一致性：

【完整情感设计】
{{mechanism_content}}

验证维度：

### 角色情感真实性
- 每个角色的情感反应是否**符合其背景**？
- 是否有角色在不同场景中表现矛盾（不是角色分裂，是更复杂）？
- 玩家是否能**预测**一个角色在特定情况下会如何反应？

### 关系网络平衡
- 是否所有关系对都有**足够的情感张力**？
- 是否存在"被孤立"的关系？（某个角色与其他角色都没有情感连接）
- 核心关系对是否在整个游戏中**持续发挥作用**？

### 情感节奏验证
- 开场是否足够**吸引情感注意力**？（不能只是背景介绍）
- 中盘是否有**情感升级**？（比开场更强的情感冲击）
- 结局前是否有**情感抉择点**？（让玩家感觉自己的选择有意义）

### 沉浸度验证
- 是否有"**情感脱锚点**"？（玩家会突然出戏的时刻）
- 线索/证据是否会**打断情感体验**？（对于情感本这是个问题）
- DM手册中的情感引导是否足够？

对于每个问题，给出：
- 问题描述
- 严重程度（P1/P2/P3）
- 具体修复建议
```

## Q4：结局情感设计

```
基于以下已完成的情感设计，设计{{char_count}}个结局方向：

【完整情感设计 + 一致性报告】
{{mechanism_and_balance}}

每个结局需包含：
- **结局名称** + 一句话概括
- **情感结局**：玩家走出房间时的情绪是什么？
- **角色命运**：每位角色在结局中的情感状态
- **与核心情感伤口的关系**：这个结局是否回应了角色最深的情感伤口？

结局类型参考：
- 团圆型：情感和解
- 悲剧型：情感伤口无法愈合
- 开放型：玩家自行解读
- 反转型：情感预设被颠覆

格式：每个结局一节，400字以内
```
