# 执行记录目录

此目录用于记录项目执行过程中的遇到的问题和解决方案。

## 使用方式

每次执行时，在本目录创建新的记录文件，命名格式：

```
YYYY-MM-DD_执行内容简述.md
```

例如：
- `2026-03-30_创建项目结构.md`
- `2026-03-31_机制设计第一次尝试.md`
- `2026-04-01_剧本创作第一天.md`

## 记录模板

```markdown
# 执行记录：YYYY-MM-DD

## 任务
[本次执行的任务]

## 完成内容
- [列出完成的内容]

## 遇到的问题
- [问题1]
- [问题2]

## 解决方案
- [解决方案1]
- [解决方案2]

## 下一步
- [待办事项]
```

## 执行清单

每次使用项目前，先更新 `templates/项目执行清单.md` 中的进度。
