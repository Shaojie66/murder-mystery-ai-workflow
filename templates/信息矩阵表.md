## 图例

| 符号 | 含义 |
|------|------|
| ■ | 知道全部：完整经历此事 |
| ▤ | 知道部分：只经历此事的一部分 |
| □ | 知道版本X：经历此事但认知被误导/不完整 |
| × | 不知道：此事发生时不在场/无感知 |
| ? | 不确定：可能知道也可能不知道 |

## 盲区设计示例

```
当角色A必须提到某事件但不确定细节时，使用：

□ "我听说..." （传言）
□ "我似乎看到..." （模糊记忆）
□ "我不确定是不是..." （怀疑）
□ "事后我才知道..." （事后才知道）

避免：
■ "B明明在场" （确定性描述）
■ "我当时就知道..." （直接确认）
```
